hArduino

This project is still WIP, implementing most of the Firmata protocol, but not quite everything yet. Some example programs:

  • Hello world: The standard led-blinking example: Blink.hs
  • Controlling a digital switch: Button.hs
  • Implementing a counter via a pair of push-buttons: Counter.hs
  • Reading an analog input value: Analog.hs
  • Control an LCD display: LCD.hs

You can explore the current API on Hackage: hArduino.

Patches, suggestions, and improvements are most welcome!
