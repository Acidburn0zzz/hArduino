hArduino

This project is still very much WIP, implementing only parts of the Firmata protocol. Some example programs:

  • Hello world: The standard led-blinking example: Blink.hs
  • Controlling a digital switch: Switch.hs
  • Implementing a counter via a pair of push-buttons: Counter.hs

You can explore the current API on Hackage: hArduino.

Patches, suggestions, and improvements are most welcome!
