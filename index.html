hArduino

This project is still very much WIP. Currently:

  • It can identify if an Arduino UNO is connected via a USB port.
  • You can blink the led on the Arduino Uno. See: Blink.hs
  • Explore the API on Hackage: hArduino

Note that most of the Firmata protocol is yet to be implemented. It has just about enough meat to blink the led on the Arduino! I expect to build the library over time to cover all of the protocol: Patches, suggestions, and improvements are most welcome!
