hArduino

hArduino allows Haskell programs to control Arduino boards and peripherals, using the Firmata protocol.

On Hackage (v0.6): hArduino.

Some example programs:

  • Hello world: The standard led-blinking example: Blink.hs
  • Controlling a digital switch: Button.hs
  • Implementing a counter via a pair of push-buttons: Counter.hs
  • Reading an analog input value: Analog.hs
  • Control an LCD display: LCD.hs
  • Sending and receiving pulses: Pulse.hs
  • Measure distance using HC-SR04: Distance.hs
  • Control a 7-segment-display: SevenSegment.hs
  • Control a servo motor: Servo.hs

NB. Sample programs above use the device /dev/cu.usbmodemfd131 to access the Arduino, which happens to be the port I use on my development Mac machine. Make sure you tweak this string properly, pointing to the USB port your Arduino is connected to. (On Linux, this will be /dev/ttyUSB* or similar. On Windows, it will look something like COM3.)

There is a short video (4m29s) demonstrating the blink example.

hArduino has been tested on Mac OSX and Windows 7. Patches, bug reports, and suggestions are most welcome.

Release notes.

Arduino setup

To use hArduino, you need to make sure your Arduino board is connected to your computer, and it is running the "StandardFirmata.ino" program that comes with the Arduino tool-kit:

  • Install the Arduino environment: http://arduino.cc
  • Download the "StandardFirmata.ino" on to your Arduino, using the Arduino environment. See http://firmata.org for details.
  • [Optional] To make sure everything is in order, you can run the standard test program to ensure your computer can access the Arduino via Firmata: firmata_test

Note that you need to do these steps only once. Once you get StandardFirmata running on your board, it will stay there even if you disconnect your Arduino from the USB port, or completely power it down.
